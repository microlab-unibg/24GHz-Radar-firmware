S2GLP_Pulsed_Doppler

===============================================================================
Description of project
===============================================================================
Doppler Application Firmware for Sense2GoL Pulse demo board.

Features:
    - Raw data acquisition for Doppler supported
    - Standard Doppler algorithm
    - Control BGT and Sample and hold unit
    - Radar GUI support 

===============================================================================
Used Hardware                               
===============================================================================
This Pulsed Doppler Application Firmware can be tested using the following hardware platforms:
    - RBB_XMC_4700 V2.0
    - Sense2GoL Pulse BGT24LTR11 Shield V1.0 or 
      Sense2GoL Pulse BGT24LTR11 Shield V3.0
