===============================================================================
Folder Contents
===============================================================================

- Binary
  *  Precompiled FW packages that directly can be used for flashing.
  
- Communication Library
  *  C/C++ and MATLAB communication interface.

- DAVE project
  *  FW & SW source code in from of DAVE™4 projects.

- XMC_Serial_Driver
  *  Windows COM Port driver to be installed once to be able to talk with the device via a serial connection.

- Radar_GUI_Configs
  *  Radar GUI XML-based configuration files, for Default settings, Low Power settings and High Performance settings.

- Power_Consumption_Calculator
  *  MATLAB script to calculate the power consumption for provided and individual device settings.
